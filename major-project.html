Major Project - Final year (Text Only Atm)

For my Final Year I decided to Build a CSS Animation Previewer, which would allow the user to do the following:

  • Select from a range of CSS animations
  • Apply them to an object (whether it be Text or Shape based)
  • Change the duration
  • Download or copy the CSS Code
  • Save said animation to a list within their profile.

Project Goals / Objectives

I took on this idea as the project to see how far I could push myself and grow in regards to CSS Animations, JavaScript, PHP and the ability to Design around those to create an application that I could be proud of.
