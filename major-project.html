Major Project - Final year

Visit the App

For my Final Year I decided to Build a CSS Animation Previewer, which would allow the user to do the following:

  • Select from a range of CSS animations
  • Apply them to an object (whether it be Text or Shape based)
  • Change the duration
  • Download or copy the CSS Code
  • Save said animation to a list within their profile.

The Goal

The main goal was to further increase my understanding of JavaScript and CSS Animations, how they work, how far can I go and how much can I learn. I wanted to build an interactive application that users could play around with,

Initial Sketches

First things first. Got to sketch out some ideas to get a sense of how layout / UX could possibly be and give me some direction of where to go from here.


Product Workflow

Laying out how you want the User Experience to flow is very useful in order to really visualize and build a foundation of how you want your application to be navigated.

As you can see, it really gives you a skeletal structure of how the application will function at release.
